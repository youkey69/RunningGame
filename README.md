# Pixel Marathon Runner

スマホ操作を優先した、ミニマム構成の横スクロール型マラソンアクションRPGです。依存パッケージなしで `index.html` をブラウザで開くだけで遊べます。

## 遊び方

- 5,000m先のゴールを目指します。
- 走るとスタミナが減り、0になるとゲームオーバーです。
- 紙コップを取るとスタミナが50%回復します。
- 敵ランナーにぶつかるとスタミナが減り、ペースが低下します。
- 速くゴールするほど賞金Gが増えます。

## 操作

| 操作 | スマホ | PC |
| --- | --- | --- |
| 奥のレーンへ移動 | 画面中央より上をタップ | 上矢印 |
| 手前のレーンへ移動 | 画面中央より下をタップ | 下矢印 |
| 減速 | 画面左端をダブルタップ | 左矢印 |
| 加速 | 画面右端をダブルタップ | 右矢印 |

## 起動

```bash
python3 -m http.server 8000
```

その後、ブラウザで `http://localhost:8000` を開いてください。
